# 📊 Análisis de Calidad de Alimentos

## 🎯 Objetivo
Analizar incidencias en productos alimenticios (pescado, carne y pollo) para identificar riesgos en conservación y manipulación.

---

## 📂 Dataset
El dataset contiene información sobre:
- Fecha  
- Producto  
- Temperatura  
- Lote  
- Incidencia  
- Tipo de problema  

---

## 🧹 Limpieza de datos
- Conversión de fechas a formato datetime  
- Revisión de valores nulos  

---

## 📊 Análisis exploratorio
- Incidencias por producto  
- Tipos de problemas más frecuentes  
- Relación entre producto y tipo de problema  

---

## 📈 Visualizaciones
Gráficos para identificar patrones de incidencias por producto.

---

## 🔍 Insights clave
- El pescado presenta incidencias constantes → problema estructural  
- La carne presenta contaminación → mayor riesgo sanitario  
- El pollo muestra un aumento progresivo de incidencias  

---

## ✅ Recomendaciones
- Mejorar control de temperatura en pescado  
- Reforzar higiene en carne  
- Supervisar almacenamiento en pollo  

---

## 🛠️ Tecnologías usadas
- Python  
- Pandas  
- Matplotlib
